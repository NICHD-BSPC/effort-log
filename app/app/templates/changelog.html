Changelog

2022-12-15

  • Implemented NIH login via LDAP to address security compliance. This also now checks the "personnel" field in the entry form to ensure it's the logged-in user. Admin users (with AA accounts) can modify anyone's entries.

2021-04-09

  • Added option to color by project, PI, or person

2021-04-08

  • Add new plotting functionality. By default, all personnel's effort are shown for the past week. Choose a name from the dropdown to show the effort for just that person, and use the range slider on the bottom to select a different time interval.
  • Add a separate page for the changelog
  • Added option to show/hide weekends in the plot (default is to hide them)
  • Sort table by ID by default, so that most recently-added entries are at the top.

2021-04-03

  • Fix dates. Previously when adding a new entry, no matter what date you used it always changed to the current date and you had to edit the entry later. This has been fixed, and the date you set will be correctly used.
  • Table is sorted by date by default.
