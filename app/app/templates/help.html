Help

  • This app is designed to make it as easy as possible to enter what you worked on during the day.
  • If a PI or project doesn't exist, let Ryan know on Slack. This is probably because a project has been created recently but the config for this app has not been updated.
  • If you worked on multiple projects in a day create entries for each of them. Note that the total fraction for the day cannot exceed 1.0.
  • A day does not need to sum to 1.0. If you had meetings, went to talks, workshops, training, etc then this does not need to be entered.
  • If you worked on a core project, that should be entered.
  • If you worked on something that is core-related but not explicitly a project, choose core -> other.
